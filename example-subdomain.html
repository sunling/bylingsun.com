如何在子域名中使用共享组件

这是一个示例页面，展示如何在子域名（如 cards.bysunling.com, meetup.bysunling.com, blog.bysunling.com）中使用共享的 header 和 footer 组件。

使用步骤：

  1. 推荐方式：在 HTML 的 <head> 中引入动态加载器：
    <script src="/scripts/dynamic-loader.js"></script>
    这会自动检测环境并加载正确的CSS和JS文件
  2. 手动方式：分别引入CSS和JS：
    CSS: <link rel="stylesheet" href="https://bysunling.com/styles/shared.css">
    JS: <script src="https://bysunling.com/scripts/shared.js"></script>
  3. 确保 body 标签使用 flex 布局，让 header 和 footer 能正确定位
  4. 如果需要自定义 header 或 footer，可以添加 class="custom-header" 或 class="custom-footer" 来阻止自动插入

功能特性：

  • 🔄 自动环境检测：本地开发和生产环境自动切换
  • 🔐 自动检测用户登录状态
  • 👤 已登录用户显示用户名和退出按钮
  • 🔗 未登录用户显示登录/注册链接
  • 📱 响应式设计，适配移动端
  • 🎨 统一的视觉风格
  • 🌐 跨域名认证状态同步

环境支持：

  • 本地开发：localhost, 127.0.0.1, 192.168.x.x, 10.x.x.x, *.local
  • 生产环境：所有其他域名，自动使用 https://bysunling.com 作为资源源
